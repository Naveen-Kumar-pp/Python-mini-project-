# Bank Account Simulation (Python)

A menu-driven Bank Account simulation built using Python and Object-Oriented Programming.

## Features
- Deposit money
- Withdraw money
- Check account balance
- Input validation
- Exception handling
- Clean modular structure

## Concepts Used
- OOP (Classes and Objects)
- Constructor (__init__)
- Instance variables
- Loops and conditionals
- Exception handling
- Program structure best practices

## How to Run
python bank_account.py
